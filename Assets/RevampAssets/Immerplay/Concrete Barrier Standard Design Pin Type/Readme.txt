This package includes the assets for the Standard, URP, and HDRP pipelines, the default assets included are for the Standard pipeline however you can install the URP or HDRP assets by importing the assets using the packages included in the URP and HDRP folders. You may then delete the folders containing the pipelines you do not require without any impact on the functionality of this asset. E.g. if you are using URP, you may delete the Standard and HDRP folders to save on project space.
